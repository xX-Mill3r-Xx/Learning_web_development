Hiperligaçoes

Presença constante nas paginas Web

As hiperligações ou links em HTML permitem que você crie um elemento clicável que leva o usuário a outra página da web, seção da página, arquivo ou recurso. O elemento para criar uma hiperligação é a tag e é um dos elementos mais comuns em HTML. Aqui está um exemplo básico de como criar um link em HTML:


Clique aqui para ir para o site exemplo

Neste exemplo, a tag -a- é usada para criar um link e a URL para o site exemplo é especificada no atributo href. Quando o usuário clica no texto "Clique aqui para ir para o site exemplo", o navegador irá carregar o site exemplo.
Além de especificar URLs, os links também podem ser usados para seções de uma página, arquivos locais, e-mails e muito mais. Para especificar uma seção de uma página, você pode usar a sintaxe #nome-da-seção no atributo href. Para especificar um arquivo local, basta inserir o caminho para o arquivo em relação ao diretório atual. Por exemplo, se o arquivo estiver na mesma pasta que a página atual, você pode usar href="arquivo.html" para criar um link para ele.
Existem muitas outras opções e atributos que você pode usar com a tag -a- para personalizar o comportamento do link, incluindo o atributo target para especificar onde o link deve ser aberto (por exemplo, em uma nova janela ou na mesma janela), o atributo title para fornecer um texto de dica quando o usuário passa o mouse sobre o link, e assim por diante.
